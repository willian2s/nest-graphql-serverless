## Description

Application developed with Nest, GraphQL, MongoDB e AWS Lambda

## Installation

```bash
$ yarn global add serverless

$ yarn install
```

## Running the app

```bash
# serveless mode
$ yarn start:sls

# development mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
